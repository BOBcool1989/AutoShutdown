# ⏻ 自动关机助手

一款简洁美观的 Windows 自动关机工具。设置关机时间后自动最小化到系统托盘静默运行，到达设定时间后弹出确认对话框——点击「跳过本次」仅取消这一次，定时任务保留到明天同一时间继续生效。

## ✨ 功能特性

- **定时关机** — 精确设置关机时间（时:分），支持跨天
- **快捷设置** — 一键设置 30分钟后 / 1小时后 / 2小时后 关机
- **实时倒计时** — 大字体倒计时显示 + 进度条
- **弹窗确认** — 到达设定时间后弹出确认对话框，60秒倒计时
  - 「跳过本次」— 仅跳过这一次，定时任务保留，明天同一时间再次提醒
  - 「立即关机」— 不再等待，立刻执行关机
- **彻底取消** — 主界面点击「取消关机」才会完全停止定时任务
- **系统托盘** — 开始后自动最小化到系统托盘，右键菜单可操作
- **开机自启** — 可选开机自动启动并最小化到托盘
- **配置保存** — 自动记住上次设置的关机时间和偏好
- **深色主题** — 精美深色界面，护眼舒适
- **兼容性强** — 支持 Windows 7 / 8 / 10 / 11 及更高版本

## 🔑 核心逻辑

```
设置关机时间 → 开始倒计时 → 最小化到托盘
                               ↓
                      到达设定时间 → 弹出确认对话框
                               ↓
                    ┌──── 跳过本次 ────┐
                    ↓                  ↓
              顺延到明天同一时间     立即关机
              继续倒计时等待
                    ↓
               明天再次提醒...

主界面「取消关机」= 彻底停止定时任务
```

## 📸 界面说明

### 主界面
- 深色卡片式布局
- 时间选择器 + 快捷按钮
- 实时倒计时 + 进度条
- 「开始定时关机」/ 「取消关机」

### 关机确认对话框
- 到达设定时间后自动弹出并置顶
- 60秒倒计时，超时自动关机
- 「跳过本次」— 保留定时，明天再提醒
- 「立即关机」— 立刻执行

### 系统托盘
- 自动最小化到系统托盘
- 左键双击：显示主窗口
- 右键菜单：显示窗口 / 取消关机 / 退出

## 🚀 使用方法

### 方式一：运行可执行文件（推荐）

直接下载 [AutoShutdown.exe](https://github.com/BOBcool1989/AutoShutdown/releases/latest) ，双击运行即可，无需安装任何环境。

### 方式二：从源码运行

```bash
pip install pystray Pillow
python auto_shutdown.py
```

### 操作说明

1. 设置关机时间（时:分），或使用快捷按钮
2. 点击「▶ 开始定时关机」
3. 程序自动最小化到系统托盘
4. 到达设定时间后，弹出确认对话框
5. 点击「跳过本次」仅取消这一次，明天同一时间再提醒
6. 如需彻底取消，在主界面点击「✖ 取消关机」

## ⚙ 设置选项

| 选项 | 说明 |
|------|------|
| 开机自动启动 | 勾选后开机自动启动并最小化到托盘 |
| 开始后最小化到托盘 | 勾选后点击开始自动最小化到托盘 |

## 🔧 技术说明

- **开发语言**：Python 3 + tkinter
- **系统托盘**：pystray + Pillow
- **开机自启**：Windows 注册表（HKCU\Run）
- **配置存储**：%APPDATA%\AutoShutdown\config.json
- **关机执行**：调用 Windows 原生 `shutdown` 命令，兼容所有版本
- **打包工具**：PyInstaller 单文件打包

## 💻 系统要求

- Windows 7 / 8 / 10 / 11
- 无需安装任何运行库（使用可执行文件版本）
