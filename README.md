# 自动关机助手

一款简洁美观的 Windows 自动关机工具，支持设置定时关机，并在关机前弹出确认对话框，让您有机会取消关机。

## 功能特性

- **定时关机**：精确设置关机时间（时:分）
- **快捷设置**：一键设置 30分钟后 / 1小时后 / 2小时后 关机
- **倒计时显示**：实时显示距离关机的剩余时间
- **关机确认**：关机前 30 秒自动弹出确认对话框，点击取消即可终止关机
- **界面美观**：采用深色主题设计，界面简洁大方
- **兼容性强**：支持 Windows 7/8/10/11 及更高版本

## 使用方法

### 方式一：直接运行（需要 Python 环境）

```bash
python auto_shutdown.py
```

### 方式二：运行可执行文件（推荐）

直接双击运行 `AutoShutdown.exe`，无需安装 Python。

### 操作说明

1. 设置关机时间（时:分），或使用快捷按钮
2. 点击「开始定时关机」按钮
3. 倒计时开始后，可以在任意时间点击「取消关机」
4. 关机前 30 秒会自动弹出确认对话框，选择「是」即可取消关机

## 系统要求

- Windows 7 / 8 / 10 / 11
- 无需安装任何运行库（使用独立打包版本）

## 技术说明

- 使用 Python 3 + tkinter 开发，零第三方依赖
- 调用 Windows 原生 `shutdown` 命令，兼容所有 Windows 版本
- 使用 PyInstaller 打包为独立可执行文件
