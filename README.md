# ⏻ 自动关机助手

一款简洁美观的 Windows 自动关机工具。设置关机时间后自动最小化到系统托盘静默运行，到达设定时间后弹出确认对话框，可取消关机。

![版本](https://img.shields.io/badge/版本-v2.0.0-e94560)
![平台](https://img.shields.io/badge/平台-Windows%207%2B-0abde3)
![协议](https://img.shields.io/badge/协议-MIT-00b894)

## ✨ 功能特性

- **定时关机** — 精确设置关机时间（时:分），支持跨天
- **快捷设置** — 一键设置 30分钟后 / 1小时后 / 2小时后 关机
- **实时倒计时** — 大字体倒计时显示 + 进度条
- **关机确认** — 到达设定时间后弹出确认对话框，60秒倒计时，点击「取消关机」即可终止
- **系统托盘** — 开始后自动最小化到系统托盘，右键菜单可操作
- **开机自启** — 可选开机自动启动并最小化到托盘
- **配置保存** — 自动记住上次设置的关机时间和偏好
- **深色主题** — 精美深色界面，护眼舒适
- **兼容性强** — 支持 Windows 7 / 8 / 10 / 11 及更高版本

## 📸 界面预览

### 主界面
- 深色卡片式布局
- 时间选择器 + 快捷按钮
- 实时倒计时 + 进度条
- 一键开始/取消

### 关机确认对话框
- 到达设定时间后自动弹出
- 60秒倒计时，超时自动关机
- 可点击「取消关机」或「立即关机」

### 系统托盘
- 自动最小化到系统托盘
- 左键双击：显示主窗口
- 右键菜单：显示窗口 / 取消关机 / 退出

## 🚀 使用方法

### 方式一：运行可执行文件（推荐）

直接下载 [AutoShutdown.exe](https://github.com/BOBcool1989/AutoShutdown/releases/latest) ，双击运行即可，无需安装任何环境。

### 方式二：从源码运行

```bash
pip install pystray Pillow
python auto_shutdown.py
```

### 操作说明

1. 设置关机时间（时:分），或使用快捷按钮
2. 点击「▶ 开始定时关机」
3. 程序自动最小化到系统托盘
4. 到达设定时间后，弹出确认对话框
5. 点击「取消关机」可终止，否则60秒后自动关机
6. 随时可从托盘右键「取消关机」

## ⚙ 设置选项

| 选项 | 说明 |
|------|------|
| 开机自动启动 | 勾选后开机自动启动并最小化到托盘 |
| 开始后最小化到托盘 | 勾选后点击开始自动最小化到托盘 |

## 🔧 技术说明

- **开发语言**：Python 3 + tkinter
- **系统托盘**：pystray + Pillow
- **开机自启**：Windows 注册表（HKCU\Run）
- **配置存储**：%APPDATA%\AutoShutdown\config.json
- **关机执行**：调用 Windows 原生 `shutdown` 命令，兼容所有版本
- **打包工具**：PyInstaller 单文件打包
